For Employees
Register with name + email → receive a 6-digit OTP code by email to verify identity
Fill in a structured annual review form covering:
Personal details & job info
Satisfaction & retrospective
Training history review
Skills assessment (technical + soft skills)
Career perspectives & development needs
Summary comments
Save as draft and come back later
Submit to HR when ready
Get notified (in-app + email) when approved or sent back for corrections
Download their completed form as Word (.docx) or PDF
For Managers / HR
Register with a secret manager code → OTP email verification
Dashboard showing all employee submissions with status badges
Filter by status: Draft / Pending Review / Approved / Needs Correction
Click into any submission to review the full form
Approve or Reject with feedback — employee is notified automatically by email
Download any form as Word or PDF
When approved: PDF is automatically uploaded to SharePoint in the EDP_Approved folder
Database Admin tab to:
See summary stats
Export all data as CSV or JSON
Remove employees or managers
Reset the entire database
