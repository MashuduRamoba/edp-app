EDP 2026 — Setup Guide Windows

Follow these steps to get the app running on your computer. No coding experience needed.

1

Download & Install Node.js

Node.js is a free tool that lets your computer run the app. You only install it once.

  1. Open your browser and go to https://nodejs.org
  2. Click the big green button labelled "LTS (Recommended)"
  3. Open the downloaded file (e.g. node-v20.x.x-x64.msi)
  4. Click Next → Next → Install (keep all defaults)
  5. When finished, click Finish
💡 How to check it worked: Press Windows key + R, type cmd, press Enter. In the black window type node --version and press Enter. You should see something like v20.11.0
2

Extract the EDP App Folder

You should have received a file called edp-app.zip.

  1. Find edp-app.zip in your Downloads folder
  2. Right-click it → "Extract All…"
  3. Choose a location (e.g. your Desktop or Documents)
  4. Click Extract

After extracting you should see a folder with this structure:

📁 edp-app/ ├── START-APP.bat ← double-click this to launch ├── package.json ├── 📁 public/ │ └── index.html └── 📁 src/ ├── index.js └── App.jsx ← the EDP form app
3

Launch the App

This is the only step you need to repeat every time you want to use the app.

  1. Open the edp-app folder
  2. Double-click the file called START-APP.bat
  3. A black command window will open — this is normal, don't close it
  4. After ~30 seconds, your browser will automatically open at http://localhost:3000
⚠️ First launch only: The first time you run it, it will download packages (~2–3 minutes). After that it starts in under 10 seconds.
💡 To stop the app: Click the black command window and press Ctrl + C, then close it.
4

Using the App

👤 For Employees

  • Click "Register" the first time to create your account
  • Next time, enter your email under "Returning Employee"
  • Fill in your EDP form, save drafts anytime
  • Click "Submit for Review" when ready
  • You'll get a notification when HR approves or returns it
  • If returned: read the manager's note, fix the form, resubmit
  • Once approved: download your form as a .docx to upload to SharePoint

5

Sharing with Other Employees Optional

Right now the app runs only on your computer. To let other employees access it from their own computers, you have two options:

Option A — Share on your local network (free, quick)

If everyone is on the same office Wi-Fi/network:

  1. Find your computer's IP address: press Win+R, type cmd, type ipconfig
  2. Look for IPv4 Address (e.g. 192.168.1.45)
  3. Other employees open their browser and go to http://192.168.1.45:3000
⚠️ Your computer must be on and the app must be running for this to work.

Option B — Deploy online (free hosting)

For permanent access from anywhere, the app can be deployed to Vercel or Netlify (both free). Ask your IT team or let us know and we'll guide you through it.

Troubleshooting

  • "node is not recognized" → Node.js wasn't installed correctly. Restart your computer and try Step 1 again.
  • Browser doesn't open automatically → Manually open your browser and go to http://localhost:3000
  • "Something is already running on port 3000" → Type Y and press Enter when prompted.
  • App is slow on first load → Normal! It compiles the first time. Subsequent loads are fast.
  • Data is lost after closing → Data is stored in your browser's local storage. Don't clear browser data.
